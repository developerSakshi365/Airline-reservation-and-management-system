## ğŸ§¾ Description
The **Flight Reservation and Management System** is a Java-based web application designed to simplify flight booking and management.  
It enables users to register, log in, search and book flights, while admins can manage flight details and view user messages.  
The application also integrates the **Jakarta Mail API** to send email notifications after flight bookings or cancellations.
